# ドキドキあみだくじ（出来レース演出）

動画向けの演出付きあみだくじサイトです。  
裏設定で当選者を1人固定しつつ、視聴者には最後まで分かりづらい進行で見せられます。

## 機能
- 参加者（2〜10人）を入力してあみだを自動生成
- 隠しコマンド/隠し操作で `裏設定` を解放
- `裏設定` で固定当選者を指定
- ルートを1人ずつ追跡するアニメーション
- 当選者を最後に回す演出（ON/OFF可）
- フェイクスポットライト演出
- 当選時のフラッシュ + 紙吹雪
- フッター表記: `by 定例会幹事`
- 結果は `当たり` 1つ + それ以外すべて `ハズレ`

## 使い方
1. `index.html` を開く
2. 名前を入力（1行に1名）
3. 裏設定を開く場合はどちらかを実行
   - キーボードで `uraopen` または `/admin` を入力
   - 画面下 `by 定例会幹事` を1秒以内に3回クリック
4. 必要なら `裏設定` から固定当選者を選択
5. `あみだ生成` を押す
6. `演出スタート` で抽選演出開始

## GitHub Pages で公開
1. このフォルダを GitHub リポジトリに push
2. GitHub の `Settings` → `Pages` を開く
3. `Build and deployment` の `Source` を `Deploy from a branch` に設定
4. Branch を `main`（または使うブランチ）/`root` にして保存
5. 数分待つと公開URLが発行されます

## ファイル構成
- `index.html` : 画面構成
- `style.css` : デザインとアニメーション
- `script.js` : あみだ生成・出来レース制御・演出ロジック
